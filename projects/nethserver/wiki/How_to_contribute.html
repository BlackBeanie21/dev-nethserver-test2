How to contribute¶

NethServer welcomes contributions!

To contribute a new package, or a fix to an existing package, use the following procedure:
  1. Create an account on GitHub
  2. Register on http://community.nethserver.org/
  3. Make changes and commit them to your fork. See Build RPMS for details on setting up a good build environment
  4. Submit a pull request when your contribution is ready
