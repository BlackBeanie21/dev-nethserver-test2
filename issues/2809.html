Category:nethserver-firewall-base

Description

Create a new interface module for managing policy routing rules.

Using the UI, the user should be able to:
  • Create/edit a rule
  • Select source and destination using object picker
  • Select service using object picker
  • Re-order the rule list
For policy routing rules format see:
  • http://docs.nethserver.o/projects/nethserver-devel/en/latest/gateway.html#force-traffic-to-a-specific-provider
