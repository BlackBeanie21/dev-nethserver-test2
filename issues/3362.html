Adagios: automatic IP configuration for nsclient

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-adagios
Target version:v6.7
Resolution: NEEDINFO:No

Description

Adagios permits to create agent bundle (nsclient++) to be installed inside Windows machines.
The default configuration for nsclient allows connections only from localhost.

As an improvement, the system should auto-configure nsclient to accept connections from all server's green IPs.
This can be done by creating a template for this file: /usr/share/okconfig/client/windows/nsclient/datafiles/allowed_hosts.ini

Associated revisions
