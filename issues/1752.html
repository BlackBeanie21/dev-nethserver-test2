Description

When installed, nethserver-gateway package provides a UI (user interface) to assign one of the following roles to a specific network interface:
  • green (also assigned by bootstrap-console
  • red
  • blue
  • orange
The same UI module sets IP configuration for each interface.
  • Red interfaces have Static or DHCP modes
  • other interfaces have only Static mode

The SystemMode property is no more used in configuration templates, but is kept to quickly check the server configuration by admins and monitoring tools (ardad).
