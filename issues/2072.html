#3

Added Bypass property to squid key. The new property contains a list of ip address which will be excluded from transparent proxy.

Example:

squid=service
    Bypass=192.168.1.22,192.168.1.24
    Mode=transparent
    NoCache=
    ParentProxy=
    Runlevels=2,3,4,5
    TCPPort=3128
    TCPPorts=3128,3129,3130
    status=enabled
