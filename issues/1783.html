Fix¶

To update the packages type as root:

   # yum --disablerepo=centos-updates update

The problem is probably originated by build-iso command in nethserver-devbox package.

During nethserver-iso-x86_64.cfg (python) evaluation, os.environ returns a cleaned up environment, so the exported MOCK_RELEASEVER variable, set by build-iso command is not available.
