Description

Wan latency monitoring can be obtained setting PingHosts to relevant addresses, info here:
http://docs.nethserver.o/projects/nethserver-devel/en/latest/collectd.html#ping-plugin

When multiwan is is active, wan link monitor (lsm) automatically obtains latency to the ip used to check link availability.

I'd like to automatically collect latency information with collectd through lsm.
