Replace source:nethserver-firewall-base|root/usr/share/nethesis/NethServer/Module/NetworkAdapter.php@f9ebff4b#L39 with udev rule like this:<pre>
# cat /etc/udev/rules.d/90-NetworksDB.rules
SUBSYSTEM=="net", KERNEL=="eth*", RUN="/usr/libexec/nethserver/update-networks-db" PROGRAM=&quot;/usr/libexec/nethserver/update-networks-db&quot;
</pre>

Note that PROGRAM can be run concurrently if the rule matches more than one eth* interface.
