Bug #2214

The problem
Using Thunderbird 17 ESR the @Shared@ shared folder was shown if the user is part of a group with the email service activated.

How To Replicate
# Create 1. create a group
# Select 2. add the interested user to the group
3. check the "Email" checkbox flag and select "Send a copy of the message to group members" radio button
# Add 4. check the interested user to the group

Check the Thunderbird folder list (of the selected user): user), as a result the folder "Shared" &quot;shared&quot; was created. created
