Feature #2509

*pflogsumm* pflogsumm --detail 10 -q /var/log/maillog
gives some statistics stats about mail. It could be shown on request on a Dashboard dashboard tab or emailed to the administrator.

"mailgraph":https://groups.google.com/d/msg/nethserver/_Hqxx_YNk4M/hFiQ7MupEX4J could be also shown in the Dashboard page

Both programs expect a maillog format where Postfix @syslog_name@ has the default value @postfix@. To adapt the maillog (see #2784) to this requirement we could use a filter, such as:<pre>

sed -r 's/(default|transfer|delivery|submission)\/(s|l)mtp/postfix\/smtp/' /var/log/maillog

</pre>

Add an administrator's manual section where the calculation criteria are explained.
